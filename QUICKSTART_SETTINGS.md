# 🎮 Game Settings - Quick Start Guide

## What's New?

All game settings are now **persistent** and **apply immediately** across all gamemodes!

---

## How to Access Settings

### In Any Gamemode
1. **Click the ⚙️ Settings button** (top-right corner)
2. **Adjust any setting** you want
3. Changes apply **instantly**
4. **Save & Close**
5. Your settings are **automatically saved**

---

## Available Settings

### 🎮 Auto-Repeat Movement
- **What:** How movement keys behave when held
- **OFF:** Tap once = move once (requires tapping)
- **ON:** Hold key = continuous movement
- **Default:** ON ✅

### 👻 Show Ghost Piece
- **What:** Preview of where your piece will land
- **OFF:** Cleaner view without preview
- **ON:** Helpful preview guide
- **Default:** ON ✅

### 📊 Show Grid Lines
- **What:** Visible grid on the game board
- **OFF:** Clean board (default)
- **ON:** Helps with alignment
- **Default:** OFF

### ✨ Particle Effects
- **What:** Animations when lines clear
- **OFF:** No animations (faster/cleaner)
- **ON:** Visual feedback animation
- **Default:** ON ✅

---

## Where It Works

✅ Classic Mode (Traditional Tetris)
✅ Speedrun Mode (Target Destruction)
✅ Extra Mode (New Pieces)
✅ Versus Mode (Head-to-Head)
  - Both Player 1 and Player 2 respect all settings

---

## Reset Settings

**Click "Reset to Defaults"** in the Settings dialog to restore all settings to their original values.

---

## Where Are My Settings Saved?

✅ **Browser Storage** (localStorage)
- Persists across browser sessions
- ~200 bytes of data
- Restored automatically when you return

**Settings are NOT saved if:**
- You clear your browser history/cache
- You use private/incognito mode (temporary)

## Troubleshooting

### "My settings are not saving"
- Check browser localStorage is enabled
- Try clicking "Save & Close" explicitly
- Refresh the page and check if they persisted
- Try resetting to defaults

### "Settings changed but game didn't update"
- Your settings should apply immediately
- If not, try refreshing the page
- Check the settings dialog to confirm values

### "I want to restore default settings"
- Click "Reset to Defaults" in settings dialog
- All settings will restore to original values

### "Settings work in one mode but not another"
- This shouldn't happen (settings are global)
- Try resetting to defaults
- Contact support if issue persists

## FAQ

### Q: Do settings apply to new games only?
**A:** No, changes apply immediately to ongoing games too!

### Q: Can I have different settings per gamemode?
**A:** Currently, all gamemodes share the same settings. Settings are global.

### Q: Will my settings sync to other devices?
**A:** No, settings are stored locally in your browser only.

### Q: Can I export/import my settings?
**A:** Not yet, but it's planned for future updates!

### Q: Do audio settings work?
**A:** The UI is there, but audio integration is for future updates.

### Q: What if I want to customize controls?
**A:** Currently only auto-repeat is customizable. Full keybinding customization is planned.
