# Versus Mode Controls - Quick Reference

## 🎮 NEW CENTER CONTROLS

### Speed Selector
Control game speed for BOTH players simultaneously.

| Button | Speed | Drop Time | Best For |
|--------|-------|-----------|----------|
| **Slow** | 1500ms | 1.5 seconds | Beginners, Practice |
| **Normal** | 1000ms | 1 second | Standard gameplay |
| **Fast** | 500ms | 0.5 seconds | Advanced players, Challenge |

**How to use:**
- Click desired speed button before starting
- Speed is locked during active gameplay
- Can change speed when paused
- Both players always have same speed

**Visual indicator:**
- Active speed: Purple background with glow
- Inactive speeds: Transparent with border
- Disabled: Grayed out during gameplay

---

### Pause/Resume Button
Pause or resume both players at once.

**States:**
- **Playing** → Shows "⏸ PAUSE" button
- **Paused** → Shows "▶ RESUME" button

**How to use:**
- Click during gameplay to pause both players
- Click again to resume
- Both players pause/resume together
- Settings button also pauses the game

**Keyboard alternative:**
- Press `P` key (works for individual players)

---

### Reset Button
Reset both games and clear winner.

**How to use:**
- Click anytime during or after gameplay
- Resets both boards to starting state
- Clears winner display
- Keeps same speed setting

**Visual feedback:**
- Red hover glow effect
- Circular arrow icon
- Always available

---

### Start Battle Button
Begin the versus match.

**How to use:**
- Select desired speed first (optional)
- Click "START BATTLE" to begin
- Both players start simultaneously
- Pieces begin falling immediately

**Visual effects:**
- Pink-to-purple gradient
- Shine animation on hover
- Prominent full-width button

## ⚡ QUICK ACTIONS

### Starting a Game
1. **Choose Speed**: Click Slow, Normal, or Fast
2. **Start**: Click "START BATTLE"
3. **Play**: Use keyboard controls for your player

### During Gameplay
- **Pause**: Click pause button in center
- **Resume**: Click resume button
- **Reset**: Click reset to start over
- **Settings**: Click ⚙️ (top-right) - auto-pauses

### Changing Speed
1. **Pause** the game
2. **Click** desired speed button
3. **Resume** to continue with new speed

### Ending a Game
- **Winner appears** when one player loses
- **Play Again** button appears on winner overlay
- **Or use Reset** in center controls

## 🔧 TECHNICAL DETAILS

### Synchronization
- All controls affect **both players equally**
- Speed changes apply to both boards
- Pause/resume synchronized
- No individual player control from center panel

### Keyboard Controls (Still Available)
**Player 1 (Right):**
- Arrows: Move
- Up: Rotate
- Down: Soft drop
- Space: Hard drop
- C: Hold piece
- **P: Pause** (individual)

**Player 2 (Left):**
- A/D: Move
- W: Rotate  
- S: Soft drop
- Shift: Hard drop
- Q: Hold piece

### Settings Integration
- Settings button (⚙️) pauses both players
- All game settings apply to both players
- Settings persist across games

## 🐛 TROUBLESHOOTING

**Speed button won't click?**
- Game must be paused or not started
- Speed is locked during active gameplay

**Pause button missing?**
- Only appears during active gameplay
- Disappears before game starts or after game over

**Both players not pausing?**
- Center controls affect both players
- Use P key for individual pause (if needed)

**Reset not working?**
- Should always work - check for errors
- Refresh page if issues persist
